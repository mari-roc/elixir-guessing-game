# Elixir Guessing Game

A simple number guessing game built with Elixir and CLI interaction.

The player tries to guess a randomly generated number while receiving hints like “higher” or “lower” until the correct answer is found.

---

## Features

* Random number generation
* CLI interaction with `IO.gets`
* Recursive game loop
* Number comparison using `case`
* Higher/lower hints
* Simple and clean project structure

---

## Technologies

* Elixir

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/mari-roc/elixir-guessing-game.git
```

### Navigate to the project folder

```bash
cd elixir-guessing-game
```

### Run the project

```bash
mix run
```
